## 開発内容
- ほぼ記事を写経し、以下の機能を実装しています
  - 質問したいことをChat GPTに問いかける
  - その回答が帰ってきて、履歴として表示される
  - 履歴をクリアして、新しい問いかけができる
  - 各セッションでの、消費したクレジットが「相談料」として表示される
- オリジナルで以下を実装しました
  - ユーザーが選んだRoleによって、返答内容が変わる
